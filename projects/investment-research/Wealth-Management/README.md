# Wealth Management Project

Portfolio construction and asset allocation project focused on long-term risk-return optimization.

## Project overview
Built and optimized a €250,000 multi-asset portfolio targeting an expected annual return of ~7% with ~8% volatility.

## Methodology
- Strategic asset allocation across asset classes
- Risk-return optimization
- Monte Carlo simulations to evaluate long-term performance
- Scenario analysis and portfolio stress testing

## Objective
Design a diversified portfolio achieving attractive risk-adjusted returns over the long term.

- Wealth Management Project report
- Portfolio analysis and simulations
